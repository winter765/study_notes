## 在相应的命令后加--help参数为查询帮助

-- git init
    为当前目录建立git仓库
    
-- git add
    添加已修改的目标文件为可提交状态
    
-- git commit
    提交当前修改
    
-- git status
    检查当前文件状态（最常用的命令）
    
-- git diff
    比较两个版本的不同之处
   
-- git rm
    删除文件后git任然会跟踪该文件，使用此命令删除文件跟踪，之后不会将该文件纳入版本管理
   
-- git log
    查看提交历史

-- git config --global core.editor vim
    设置commit时使用的默认编辑器
